Carga de certificados

Herramienta de administración. Sube certificados a Firestore (colección certificates) desde una planilla, para que verificar los reconozca.
⚠ Solo para administradores. Necesitás iniciar sesión con una cuenta con rol admin para que Firebase permita la escritura.

1 · Pegá o subí la planilla

La planilla debe tener una fila de encabezados con estas columnas (el orden no importa):
code · studentName · courseTitle · hours · issueDate  (opcional: courseId, modules)
Para modules, separá los módulos con punto y coma ; dentro de la celda.
